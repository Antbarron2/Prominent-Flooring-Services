# Prominent Flooring Services Website

This is a deploy-ready Next.js website starter for Prominent Flooring Services.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`

## Deploy

Push this folder to GitHub, then import the repository into Vercel.

## Customize first

Update these items in `app/page.tsx`:
- phone number
- email
- service area
- services list
- quote form details
